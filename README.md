# Rock Paper Scissors - Java GUI

This project is a simple Rock Paper Scissors game built using Java with a graphical user interface.

## Screenshot
![Game Screenshot]
(game_screenshot.png)

## Features
- Player vs Computer gameplay
- Random Computer choices
- Interactive graphical interface
- Beginner friendly project

## Technologies Used
- Java
- Java GUI (Swing)

## Purpose
This project was created to practice Java programming and GUI development while learning how to build interactive applications.
